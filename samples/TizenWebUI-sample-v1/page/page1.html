Multi-page application

This is a multi-page boilerplate template that you can copy to build your first Tizen Web UI Framework page.
This template contains multiple "page" containers inside.

You link to internal pages by referring to the ID of the page you want to show.
For example, to link to the page with an ID of "two", my link would have a href="page2.html" in the code.
